# 視窗（二）WAV 音效檔播放器

### 專案簡介:
> 提供簡潔的圖形化介面，讓使用者能夠輕鬆選擇並播放 WAV 格式的音效檔案。系統整合了 Windows 多媒體播放功能，支援單次播放、循環播放及即時停止，並透過群組框（GroupBox）分類管理路徑選擇與播放控制，提升操作的直覺性。

### 執行說明:
> 1.選取檔案：點擊「瀏覽」按鈕開啟檔案對話框，選擇欲播放的 .wav 檔案後，路徑將自動顯示於文字方塊中。
> 2.控制操作：
>    * 播放一次：執行音效播放動作，播完即停止。
>    * 重複播放：啟動循環模式，音效將不斷重播直到使用者手動停止。
>    * 停止播放：立即中斷當前正在播放的音效。
>    * 結束程式：按下「結束程式」按鈕後，系統會自動跳出**防呆確認視窗**進行二次確認，避免因誤觸而導致程式意外關閉，確保操作的安全性。
